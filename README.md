## About

My solutions to the Advent of Code 2024.

### Built With

- TypeScript
- Launcher build with inquirer.js

## Getting Started

### Prerequisites

- pnpm 9
- node 20

### Installation

1. git clone
2. pnpm install
3. pnpm start

## Usage

- run `pnpm start`
- download your input
- Select day
- select part
- select input
